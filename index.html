    House Rules

    Generally, this is a drinking game designed for fun - so feel free to bend or ignore the rules as you see fit. These rules are just guidelines based off what we do when things don't go smoothly:

    1. 1. If you have to drink a decimal number, round it up. You're here to get drunk, after all.
    2. 2. If a vote ends with a draw, it can be decided with a coin flip.
    3. 3. An empty drink is just a full drink that has not been poured yet, as said in the Bible probably. Therefore, if you have to down an empty drink, you must pour a whole drink first.
    4. 4. It's perfectly fine (even encouraged) to drink between turns.
